# zhihudaily-vue

> A Vue.js project

使用知乎API练习VUE

## Dependencies

```js
"dependencies": {
    "babel-runtime": "^6.0.0",
    "fastclick": "^1.0.6",
    "lib-flexible": "^0.3.2",
    "swiper": "^3.3.1",
    "vue": "^1.0.21",
    "vue-resource": "^0.7.0",
    "vue-router": "^0.7.13"
  }
```

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8300
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test

# 开启本地的server,通过jsonp中转知乎API
npm server
```
